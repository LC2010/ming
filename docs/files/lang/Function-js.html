ECMA-262-5 15.3.4.5 Sets the value of func inside the function to always be the value of thisArg when the function is called.  Optionally, function arguments can be specified (arg1, arg2, etc) that will automatically be prepended to the argument list whenever func function is called.

Parameters

thisArg{Function}
arg1{optinal} {Object}
arg2{optinal} {Object}

Example

var flatFunction = bind(myFunc, myObj, arg1, arg2);
flatFunction(arg3, arg4);
