var stringify = function(value,

ECMA-262-5 15.12.3 This method produces a JSON text from a JavaScript value.  The stringify method takes a value and an optional replacer, and an optional space parameter, and returns a JSON text.  The replacer can be a function that can replace values, or an array of strings that will select the keys.  A default replacer method can be provided.  Use of the space parameter can produce text that is more easily readable.

Parameters

value{Object} any JavaScript value, usually an object or array.
replacer{optional} {Object} an optional parameter that determines how object values are stringified for objects.  It can be a function or an array of strings.
space{optional} {Object} an optional parameter that specifies the indentation of nested structures.  If it is omitted, the text will be packed without extra whitespace.  If it is a number, it will specify the number of spaces to indent at each level.  If it is a string (such as ‘\t’ or ‘&nbsp;’), it contains the characters used to indent at each level.

Returns

{String}
