Selector

轻量选择器

支持选择器

1.简单选择器

  • #id
  • .class
  • tag
  • tag.class
  • tag#id
  • *

2.子元素选择器

3.组合选择器

注意

  • 不支持 #id.class
  • 尽可能使用子元素选择器(parent>child)而非组合选择器(parent child)
  • 不支持含有通配符 * 的组合选择器
  • 其他选择器可通过filter(elem,index,nodes)回调函数间接实现
